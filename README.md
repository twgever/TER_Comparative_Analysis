## 📖 Overview

This repository contains material developed as part of a **university project** within the context of academic coursework and experimental research activities.

The project focuses on the comparison and evaluation of different computational and machine learning approaches, with particular attention to methodology, model behavior, and performance analysis.

The repository is structured as a collection of notebooks, reports, and supporting material intended for experimentation, reproducibility, and comparative evaluation.

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
cd TER_Comparative_Analysis-main
```

(Optional) Create a virtual environment:

```bash
python -m venv venv
source venv/bin/activate
```

On Windows:

```bash
venv\Scripts\activate
```

Install the required dependencies:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn torch notebook
```

---

## 🚀 Usage
